# Lemmatizer

```
./train.sh \
    path-to-ud-treebanks \
    ud-language-name \
    ud-corpus-name \
    iso639-2-lang-code \
    morphosyntax-conf-directory \
    output-directory
```

## Example:

```
./train.sh \
    ../ud-treebanks-v2.7 \
    French \
    Sequoia \
    fra \
    ../morphosyntax.conf/ \
    output/french-model
```
